# Strudel Samples

Breakbeats, a pad, and a bass texture packaged for Strudel. Load them via:

```javascript
await samples('github:ugamochi/strudel-samples')
```

Usage examples:

```javascript
s('breaks/0').loopAt(1)   // br1.wav
s('pad/0').slow(4)        // star-pad-vox pad
s('bass/0').note('c2')    // hard bass layer
```

Files in this pack:
- `br1.wav`
- `br2.wav`
- `br3.wav`
- `br4.wav`
- `br5.wav`
- `star-pad-vox.wav`
- `hard-bass.wav`
