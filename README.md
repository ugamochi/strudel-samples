# Strudel Samples

Five breakbeats packaged for Strudel. Load them via:

```javascript
await samples('github:ugamochi/strudel-samples')
```

Usage examples:

```javascript
s('breaks:0').loopAt(1)   // br1.wav
s('br:1').fit()           // br2.wav (alias bank)
s('breaks:2').scrub('<0 0.25 0.5 0.75>').fit()
```

Files in this pack:
- `br1.wav`
- `br2.wav`
- `br3.wav`
- `br4.wav`
- `br5.wav`
